How AI Tool Calls Actually Work

People assume models can search the web, call APIs, and use tools on their own. They can't. Here's what really happens — and why it matters.

1

What People Think Happens

👤
User
"Search for X"
➡
🤖
Model
Searches the web
➡
✅
Result
Answer with sources
❌ This is wrong.
Models are text generators. They can't open a browser, call an API, or execute code.
They can only output text — including text that looks like a tool call.
2

What Actually Happens (step by step)

1
🤖 THE MODEL
User asks a question. The model doesn't search anything. It outputs JSON saying "I want to call a tool":
{
  "tool_calls": [{
    "function": {
      "name": "web_search",
      "arguments": { "query": "latest news" }
    }
  }]
}
2
❓ THE GAP
The model stops here. It's done. It output some JSON and is now waiting.
It cannot execute that tool call. It has no internet access, no API keys, no runtime.
Something else has to read that JSON, actually run the tool, and send results back.
3
⚙️ THE HOST (whoever runs the model)
The host application reads the JSON, executes the actual search/API call, and sends the results back to the model as a new message. Then the model generates its final answer.
3

Who fills "The Gap" in each scenario?

ChatGPT / Claude (Cloud)

When you use ChatGPT or Claude's website, OpenAI/Anthropic fills the gap for you.

👤 You ask a question
🤖 Model outputs tool_call JSON
🏢 OpenAI's servers execute the tool
📨 Results sent back to model
✅ Model gives final answer
Works, but your data goes to their cloud.

Private Models (No Gateway)

Running Ollama/vLLM on your own servers. No one fills the gap.

👤 You ask a question
🤖 Model outputs tool_call JSON
🛑 Nobody executes the tool call
❌ Raw JSON returned to user
🚫 No audit, no safety, no controls
Data stays private, but tools don't work.
No governance at all.

Private Models + Gateway

Gateway sits between your apps and models. It fills the gap.

👤 You ask a question
🔐 Gateway: auth + policy check
🤖 Model outputs tool_call JSON
⚙️ Gateway executes the tool
🔍 Gateway scans results for PII/safety
🤖 Model gives final answer
🔗 Gateway records everything in audit chain
Data stays private. Tools work.
Full governance and audit trail.
The bottom line
Models can ask for tools — but they can't use them.
On ChatGPT, OpenAI executes them (on their cloud).
On your private models, Gateway executes them — on your network, with full governance.
